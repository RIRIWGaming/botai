# WhatsApp Bot - Full Feature Starter (Baileys) - Enhanced

Versi ini menambahkan implementasi dasar untuk:
- Integrasi OpenAI Chat (gunakan env var OPENAI_API_KEY)
- YouTube downloader (ytdl-core)
- Remove.bg integration (gunakan env var REMOVE_BG_API_KEY)
- Logo maker (menggunakan canvas, tidak butuh API)
- Instruksi menjalankan & env vars

## Env vars yang disarankan
- OPENAI_API_KEY  -> untuk fitur AI (!ai)
- REMOVE_BG_API_KEY -> untuk fitur !removebg

## Quick start
1. `npm install`
2. Set environment variables (lihat README untuk contoh)
3. `node index.js`
4. Scan QR yang muncul
